# Introduction

This game is a 2d factory/tower defense game in space, based on physics. 
(Disclaimer: I've thought a lot about the story but currently the game doesn't have a lot of features (placing buildings, drawing some stuff but that's it))

## Inspiration

- Vectorio: artstyle
- Factorio: tower defense mechanisms (expand = more/stronger monsters)
- Dyson sphere program: Star energy collection, just a great game in general
- Icaria: Programming drones, creating more drones (seems cool but haven't implemented anything like that for now)
- Excalidraw.com: a great site for sketching, but I think a game with the same artstyle + bloom and animations would look great

## Lore

The world has been destroyed by antimatter monsters ! Fortunately, a little miniaturised robot, smaller than Plank length has survived (it's between antimatter particles).

You first start with a `little matter generator`, which transforms antimatter pellets to useful materials ! The more antimatter you get, the more space you make around your base, but caution ! After a while antimatter monsters will see this anomaly, and as you may know: $E=mc²$, so antimatter + matter = big explosion !

Use strange weapons to defend against pesky anti-drones, collect some crazy resources from antimatter bases, conquer multiple universes, and (maybe?) find the root of the Cataclysm !

### Producing energy

Little energy generator: inputs matter + antimatter => energy, but in a controlled way 😅 (or is it ?)

### Gameplay (WIP)

The `little matter generator` is a device that sucks up `base antimatter pellets` and transforms them into diverse materials. 
After some while you will be able to:

- Find a planet to get resources from (they are huge... at the beginning)
- Collect energy from stars (*Dyson Sphere Program* is great)
- From black holes, and other physics processes
- Create AI, genetically modify life, expand to multiple dimensions, change universes and physical laws...
